此站名 HYSN.NET，由 Harrison 手作於 MMXXVI 年，以 HTML、CSS、JavaScript 寫就，不假任何框架之助，寄存於 GitHub Pages。其所用字體凡四——Cormorant Garamond 與 EB Garamond 擔當西文之標題與正文，Noto Serif TC 承載繁體中文，JetBrains Mono 司掌器物之名——皆取自 Google Fonts。版面以普魯士藍配古銅色為基調，置於暖米色之上，取古籍裝幀之意。部分圖示假 Devicon 之手，「我在聽」一節之播放器則借自 Spotify。至於思想之源，則謹誌於此：柏拉圖、笛卡爾、康德、胡塞爾、卡繆、薩特諸先賢——雖至今仍在門外徘徊，而其氣韻早已潛入我思之深處；亦感念 Claude 與 ChatGPT 二者，於構築此站之際多所協力。凡此種種，皆非吾所獨有；吾所能為者，不過綴合之而已。
