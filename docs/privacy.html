隱私

❦

這是一個個人網站。以下是您造訪時實際發生的事——以及不會發生的事。

本站以靜態 HTML 呈現。您的瀏覽器會在本地儲存兩項偏好設定——深淺主題與語言——兩者僅存於您的裝置上，不會傳回伺服器。除此之外，本站不使用分析工具，不設其他 Cookie，不設帳號、表單或訂閱。

至於本站所載入的第三方資源，它們各有自家的隱私實踐：

  • Google Fonts載入字型時，會向 Google 傳送您的 IP 位址與 User Agent。
  • jsDelivr提供部分技術圖示的 CDN，可能記錄 IP 位址。
  • Spotify嵌入於「我在聽」一節，若您與之互動，Spotify 將依其自家政策放置 Cookie 並記錄行為。

若您希望完全規避，建議使用擋廣告或隱私工具，或略過該節。若您對本站之處理有任何疑問或要求，請來信 harrison@hysn.net。

此頁非律師起草；若本頁所述與本站實際行為有異，以實際為準。
